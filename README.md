An automated system for extracting and categorizing key information from resumes (PDF/DOCX). It parses candidate details like name, skills, and education, and stores them in a searchable PostgreSQL database using a simple web interface built with Flask.

Technologies -

•UsedPython (spaCy for NLP, PDFPlumber for PDF extraction)

•Flask (Web framework)

•PostgreSQL (Database)# Automated-resume-parser
